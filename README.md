# FakeBiliBili

> 一款基于 MVP+RxJava+Retrofit+Dagger2+Butterknife+Fresco+MultitypeAdapter+MD 设计风格的仿 B 站客户端

### 前期准备

* 反编译资源文件
* `mtimproxy`抓取 Api
* 获得 B 站 signKey  

更多内容请看这里 [FakeBiliBili 系列 (启动篇)](http://www.jianshu.com/p/b3b9e13bd842)

### Version 1.0.1

* 架构搭建
* 分析 Api
* 完成主页界面
